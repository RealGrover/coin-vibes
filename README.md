# coin‑vibes

A collection of interactive UI snippets and code samples for the **coin‑vibes** art project.

## New examples (added)
- **Demo 1:** Simple animated coin‑vibes visualizer (HTML/JS/CSS) in `coinvibes2/prototypes/demo1/`.
- **Demo 2:** Interactive color‑swap widget showcasing the Coin Standard palette.

These examples can be opened locally via `open coin-vibes/coinvibes2/prototypes/demo1/index.html` and are included in the live site.
